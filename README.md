# GOG Archive (static site)

Public static browser for the [GOG-Archive](https://github.com/mon5termatt/GOG-Archive) catalog: game metadata, tags, magnet links, and direct download URLs from [gog-games.to](https://gog-games.to).

**This repository contains only the deployed static site** (HTML, CSS, JS, and `data/games.json`). It is updated from the private tooling repo via `scripts/publish_pages_repo.ps1`.

## GitHub Pages

1. [Settings → Pages](https://github.com/mon5termatt/GOG-Archive-pages/settings/pages)
2. **Build and deployment → Source:** Deploy from a branch
3. Branch **`main`**, folder **`/ (root)`**

Site URL: **https://mon5termatt.github.io/GOG-Archive-pages/**

## Legal note

Metadata and links are archived from third-party sources for personal reference. This site does not host game files. Respect GOG and gog-games.to terms of use.
